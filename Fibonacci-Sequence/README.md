Fibonacci Sequence (Python)

A collection of three Fibonacci programs written in Python to practice loops, recursion, functions, and exception handling.

Programs Included

1. Fibonacci Sequence (Basic)

A simple program that generates the first n Fibonacci numbers using an iterative approach.

Features

- User input
- Positive integer validation
- Quit option
- Prints Fibonacci sequence

---

2. Fibonacci Sequence (Enhanced)

An improved version with multiple output formats and better input handling.

Features

- Functions ("intro()", "body()", "main()")
- Exception handling ("try-except")
- Positive integer validation
- Quit option
- Multiple output formats:
  - Line by line
  - Python list
  - Comma-separated values

---

3. Fibonacci Sequence (Recursive)

A recursive implementation that uses a single function to perform multiple tasks.

Features

- Recursive Fibonacci generation
- Find the nth Fibonacci term
- Calculate the sum of the first n terms
- Display the first n Fibonacci numbers
- Menu-driven interface
- Exception handling
- Positive integer validation
- Quit option

# Factorial Calculator

## Description

This project demonstrates **three different ways** to calculate the factorial of a number in Python. It also handles invalid input, negative numbers, and allows the user to exit the program by typing `quit`.

---

## Versions

### mainV1.py – Basic Iterative Version

A simple beginner-friendly implementation using:
- `while` loop
- `for` loop
- `try-except`
- `break` and `continue`

### Features
- Calculates factorial using a loop.
- Handles invalid input.
- Rejects negative numbers.
- Exit using `quit`.

---

### mainV2.py – Recursive Version

This version calculates factorial using **recursion**.

### Features
- Uses a recursive function (`facGen()`).
- Demonstrates the concept of recursion.
- Handles invalid input.
- Rejects negative numbers.
- Exit using `quit`.

---

### mainV3.py – Function-Based Iterative Version

A cleaner and more modular implementation using functions.

### Features
- Uses separate functions (`intro()`, `body()`, `main()`).
- Calculates factorial using a loop.
- Better code organization.
- Handles invalid input.
- Rejects negative numbers.
- Exit using `quit`.
